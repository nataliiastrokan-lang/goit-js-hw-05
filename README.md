## 🧩 Tasks Overview

### ✅ Task 1 — Get User Names
Extracts names from an array of user objects using `map()`.

---

### ✅ Task 2 — Users with Friend
Filters users who have a specific friend using `filter()` and `includes()`.

---

### ✅ Task 3 — Sort by Friend Count
Sorts users by number of friends in descending order using `toSorted()`.

---

### ✅ Task 4 — Total Balance by Gender
Calculates total balance by gender using `filter()` and `reduce()`.

---

## 🛠️ Technologies Used

- JavaScript (ES6+)
- Array methods:
  - `map()`
  - `filter()`
  - `reduce()`
  - `toSorted()`

---

## 💡 Key Concepts

- Functional programming approach
- Method chaining
- Working with arrays of objects
- Immutability (using `toSorted()` instead of `sort()`)
